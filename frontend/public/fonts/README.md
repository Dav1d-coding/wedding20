# Расписной шрифт имён

На hero для имён используется **Lucida Calligraphy** — системный шрифт (часто есть в Windows). Семейство задаётся в `globals.css` (класс `.font-script`), с запасными вариантами для других ОС.

Папка `public/fonts/` — для любых дополнительных статических файлов при необходимости.
