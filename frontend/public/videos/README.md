# Видео для главного экрана

Положите сюда файл, например `hero.mp4`, и в `src/app/page.tsx` укажите:

```tsx
<CinematicHero mediaSrc="/videos/hero.mp4" />
```

Поддерживаются: **`.mp4`**, **`.webm`**, **`.gif`**, а также **`.png` / `.jpg`**.

Для автовоспроизведения в браузере видео должно быть **без звука** (`muted` в разметке уже включён).
